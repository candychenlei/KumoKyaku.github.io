SourceTree-git-lfs无法识别大于10mb文件解决方法

软件学习

SourceTree 使用lfs时，单个文件大于10mb时会导致无法追踪。虽然使用git lfs 查看会显示为以追踪，但是会推送失败。

解决方法：

将选项中比较设置中二进制大小限制调大即可。

至于为什么比较设置会影响lfs，可能是个BUG。
